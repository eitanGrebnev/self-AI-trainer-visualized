# self-AI-trainer-visualized
This is still in progress. This is a project for training a local LLM on your own data and being able to see the neural network change in real time. This is for either wanting to interact with your trained data or to humanize your text in your own words. 


The visualizer right now is depricated at the moment, however I'll work on reintegrating that soon. 

The current task I've been working on is a user friendly GUI and making this as accessable as possible.
